    • Mystery Code Society's 2019-2020 Season

    Classroom computers are generously provided by our location sponsor, DoSpace. A home computer is not required to participate in these courses.

    Intro To JavaScript

    Introduction to Code, prerequisite to JavaScript Video Games

    location: DoSpace

    Intro to JavaScript is a 6 week introductory coding class for students in 5th through 12th grade who are brand new to coding! In this course, students will learn the basics of coding in JavaScript: variables, functions, loops, conditionals, and callbacks.

    • Sundays, 1-4 pm, October 6 - November 10

    Sugar Cubed: JavaScript Video Games Class

    Build your own video game!

    location: DoSpace

    JavaScript Video Games is a 12 week intermediate coding class for students in 5th through 12th grade who want to build video games in JavaScript. Students will learn to build six different video games like this one and will have time to build and share games of their own!

    Intro to JavaScript (or equivalent) is a prerequisite to take this course.

    • Sundays, 1-4 pm, January 12 - March 29

    HTMelle

    Build your own website!

    location: Abrahams Branch Public Library

    HTMelle is an introductory eight week course for students in 5th - 12th grade. Students will learn to build websites with HTML and CSS.
