    Classroom computers are generously provided by our location sponsor, Abrahams Branch Library. A home computer is not required to participate in these courses.

    HTMelle

    beginning web development

    location: Omaha Public Library, Milton R. Abrahams Branch

    HTMelle is a 6 week introductory coding class for students in 7th through 12th grade who are brand new to coding! In this course, students will learn the basics of making web pages with HTML and CSS.

    • Saturdays, October 13 - November 17

    HTMelle Masters

    continuing web development

    location: Omaha Public Library, Milton R. Abrahams Branch

    HTMelle Masters is a 9 week intermediate coding class for students in 7th through 12th grade who know a little about HTML and CSS but want to learn more. Students will build on their HTML and CSS skills, learn a bit about JavaScript and DOM manipulation, and use their newly mastered skills to create their own websites.

    • Saturdays, January 12 - March 9

    Intro to Ruby

    code fundamentals in Ruby

    location: Omaha Public Library, Milton R. Abrahams Branch

    Intro to Ruby is a 6 week beginner course in code fundamentals for students in middle and high school. This course is a prerequisite to the advanced summer course on databases and web applications, SheQL.

    • Saturdays, April 6 - May 11

    SheQL

    Databases and Web Applications in Ruby/Sinatra/Postgres

    location: Omaha Public Library, Milton R. Abrahams Branch

    SheQL is an 11 week advanced course in database design and web applications for high school students and advanced middle school students. Students will build a web app in Ruby/Sinatra/Postgres with the same tools as software developers.

    • Saturdays, June 8 - August 17
